# Instagram Ad Formats & Specifications — Brandea Campaign Reference

> Updated: March 2026 | Source: Meta Business Help, Hootsuite, AdManage.ai, WASK, QuickFrame

---

## Quick Reference: 3 Must-Have Asset Sizes

| Ratio | Размер (px) | Где используется |
|---|---|---|
| **1:1** | 1080 x 1080 | Feed, Explore Home, Search, Carousel |
| **4:5** | 1080 x 1350 | Feed (portrait) — максимум экрана на мобильном |
| **9:16** | 1080 x 1920 | Stories, Reels — полноэкранный формат |

---

## 1. Feed Image Ad

| Параметр | Значение |
|---|---|
| Размер (рекомендуемый) | 1080 x 1080 (1:1) или 1080 x 1350 (4:5) |
| High-res | 1440 x 1440 (1:1) или 1440 x 1800 (4:5) |
| Минимум | 600 x 600 px |
| Aspect ratio | от 1.91:1 (landscape) до 4:5 (portrait) |
| Лучший ratio | **4:5** — занимает максимум мобильного экрана |
| Форматы | JPG, PNG |
| Макс. размер | 30 MB |
| **Primary text** | 2,200 макс / **125 рекомендуемый** (до обрезки "...more") |
| **Headline** | 40 макс / **27 рекомендуемый** |
| **Description** | 30 макс / **25 рекомендуемый** |
| Хештеги | до 30 / оптимально 5-11 |

**Важно**: текст на изображении — менее 20% площади для оптимальной доставки.

## 2. Feed Video Ad

| Параметр | Значение |
|---|---|
| Размер | 1080 x 1350 (4:5 portrait, рекомендуемый) |
| Также | 1080 x 1080 (1:1), 1080 x 608 (landscape) |
| Длительность | 1 сек — 60 мин |
| **Оптимальная длительность** | **6-15 секунд** |
| Формат | MP4, MOV |
| Кодек видео | H.264 |
| Кодек аудио | Stereo AAC, 128 kbps+ |
| Frame rate | до 30 fps |
| Макс. размер | 4 GB |
| **Primary text** | 2,200 макс / **125 рекомендуемый** |
| **Headline** | 40 макс / **27 рекомендуемый** |
| **Description** | 30 макс / **25 рекомендуемый** |

**Важно**: субтитры обязательны (большинство смотрит без звука). Первые 3 секунды — критичны для внимания.

## 3. Stories Ad (Image)

| Параметр | Значение |
|---|---|
| Размер | 1080 x 1920 (9:16) |
| High-res | 1440 x 2560 |
| Минимум | 600 x 1067 |
| Форматы | JPG, PNG |
| Макс. размер | 30 MB |
| Показ | 5 секунд |
| **Primary text** | 2,200 макс / **125 рекомендуемый** |

### Safe Zone (1080 x 1920)
```
+----------------------------------+
|    TOP: 250px — НЕ РАЗМЕЩАТЬ     |  ← профиль, username
+----------------------------------+
|                                  |
|    SAFE AREA: 1080 x 1420 px     |  ← основной контент
|    (текст, лого, ключевые        |
|     визуалы — здесь)             |
|                                  |
+----------------------------------+
|  BOTTOM: 340px — НЕ РАЗМЕЩАТЬ    |  ← CTA кнопка
+----------------------------------+
```

## 4. Stories Ad (Video)

| Параметр | Значение |
|---|---|
| Размер | 1080 x 1920 (9:16) |
| Длительность | 1 сек — 120 сек (2 мин) |
| **Оптимальная длительность** | **до 15 секунд** |
| Формат | MP4, MOV |
| Кодек | H.264, AAC stereo 128 kbps+ |
| Frame rate | до 30 fps |
| Макс. размер | 4 GB |

**Важно**: видео >15 сек разбивается на несколько карточек автоматически. Safe zone — как у Stories Image.

## 5. Reels Ad (Video)

| Параметр | Значение |
|---|---|
| Размер | 1080 x 1920 (9:16) |
| High-res | 1440 x 2560 |
| Длительность | 1 сек — 15 мин |
| **Оптимальная длительность** | **15-30 секунд** |
| Формат | MP4, MOV |
| Кодек | H.264, AAC stereo 128 kbps+ |
| Frame rate | 30 fps |
| Макс. размер | 4 GB |
| Cover photo | 420 x 654 (1:1.55) |
| **Primary text** | **72 символа** (видимый максимум) |

### Reels Image Ad (статика)
| Параметр | Значение |
|---|---|
| Размер | 1080 x 1920 (9:16) |
| Формат | JPG, PNG |
| Макс. размер | 30 MB |
| Показ | ~5 секунд |

### Safe Zone Reels
```
+-----------------------------+----+
|  TOP: 250px — username/music|    |
+-----------------------------+    |
|                             |120 |
|   SAFE AREA: ~960 x 1420   |px  |
|   (основной контент)        |    |
|                             |    |
+-----------------------------+    |
|  BOTTOM: 340px — CTA        |    |
+-----------------------------+----+
           ↑ Справа: 120px — кнопки лайк/коммент/шейр
```

**Важно**: аудио рекомендуется (Reels — sound-on среда). Реклама, похожая на нативные Reels, работает значительно лучше. С января 2026 Reels ads также показываются в бывшем Explore feed.

## 6. Explore Home Ad

| Параметр | Значение |
|---|---|
| Размер (изображение) | 1080 x 1080 (1:1, рекомендуемый) |
| Также | 1080 x 1350 (4:5) |
| Размер (видео) | 1080 x 1080 (1:1) |
| Длительность видео | 15-30 секунд |
| Форматы | JPG, PNG (фото); MP4, MOV (видео) |
| Макс. размер | 30 MB (фото) / 4 GB (видео) |
| Текст | Как у Feed ads |

**Важно**: появляется как плитка в сетке Explore среди органического контента. Квадрат (1:1) рекомендуется. Единственное оставшееся Explore-размещение с января 2026.

## 9. Carousel Ad (Feed + Stories)

### Feed Carousel
| Параметр | Значение |
|---|---|
| Карточки | 2-10 (свайп) |
| Размер карточки | 1080 x 1080 (1:1) — все карточки одинаковые |
| Также | 1080 x 1350 (4:5) — но все одинаковые |
| Форматы | JPG, PNG (фото); MP4, MOV (видео) |
| Видео длительность/карточка | 1-60 сек (рекомендуется до 15 сек) |
| Можно смешивать | Да — фото + видео в одном carousel |
| **Primary text** | 2,200 макс / **80 рекомендуемый** |
| **Headline/карточка** | 45 макс / **32 рекомендуемый** |
| **Description** | 30 макс / **18 рекомендуемый** |
| Ссылки | Каждая карточка — своя ссылка |

### Stories Carousel
| Параметр | Значение |
|---|---|
| Карточки | до 10 (первые 1-3 показываются по умолчанию) |
| Размер | 1080 x 1920 (9:16) per card |
| Фото показ | 5 сек/карточка |
| Видео длительность | до 60 сек/карточка |

## 10. Collection Ad

| Параметр | Значение |
|---|---|
| Cover (фото/видео) | 1080 x 1080 (1:1) или 1200 x 628 (1.91:1) |
| Product images | 1080 x 1080 (1:1) |
| Продукты в Feed | 3 под cover |
| Продукты в Stories | 2 |
| Макс. размер product image | 3 MB |
| **Primary text** | 2,200 макс / **125 рекомендуемый** |
| **Headline** | 40 макс / **25 рекомендуемый** |

**Важно**: при клике открывается Instant Experience (полноэкранный лендинг внутри приложения). Только мобильные. Требует подключённый каталог продуктов в Meta Commerce Manager.

## Изменение 2026: Explore → Reels

С января 2026 размещение **Instagram Explore** (внутри контента) убрано. При клике на плитку в Explore пользователь попадает в Reels-подобный viewer. Реклама Reels теперь также показывается в бывшем Explore feed. Размещение **Explore Home** (сетка) — сохранено.
