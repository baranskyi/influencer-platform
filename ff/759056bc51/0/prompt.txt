# Nano Banana Pro — Image Generation

Generate images using Google Gemini 3 Pro Image model via the `google-genai` Python SDK.

## Quick Start

Generate a single image:
```bash
~/Personal-Super-Agent/config/analytics_env/bin/python3 {baseDir}/scripts/generate.py --prompt "A pixel art robot sitting at a desk" --output "/path/to/output.png"
```

Generate multiple images in parallel:
```bash
~/Personal-Super-Agent/config/analytics_env/bin/python3 {baseDir}/scripts/generate.py \
  --prompts-json '[{"filename": "hero1.png", "prompt": "A sunset over mountains"}, {"filename": "hero2.png", "prompt": "A forest at dawn"}]' \
  --output-dir "/path/to/output/"
```

## Options

- `--prompt TEXT`: Single image prompt
- `--output PATH`: Output file path for single image (default: `./generated.png`)
- `--prompts-json JSON`: JSON array of `{"filename", "prompt"}` objects for batch generation
- `--output-dir PATH`: Output directory for batch generation
- `--style-prefix TEXT`: Shared style prefix prepended to all prompts
- `--api-key KEY`: Override GEMINI_API_KEY env variable

## Style Prefix Example

For consistent visual style across multiple images:
```bash
~/Personal-Super-Agent/config/analytics_env/bin/python3 {baseDir}/scripts/generate.py \
  --style-prefix "Pixel art style illustration on dark navy background with neon glows." \
  --prompts-json '[{"filename": "img1.png", "prompt": "Robot at desk"}, {"filename": "img2.png", "prompt": "Robot flying"}]' \
  --output-dir "./images/"
```

## Setup

1. Install dependencies (once):
```bash
~/Personal-Super-Agent/config/analytics_env/bin/pip install google-genai Pillow
```

2. Set API key:
```bash
export GEMINI_API_KEY="your-gemini-api-key"
```

Or pass via `--api-key` flag.

## Model

Uses `gemini-3-pro-image-preview` with `response_modalities=["TEXT", "IMAGE"]`.

## Notes

- Output format: PNG
- Supports parallel generation for batch requests (threaded)
- Each prompt can optionally include a style prefix for visual consistency
- Requires `google-genai` and `Pillow` Python packages
